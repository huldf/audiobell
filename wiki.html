Informations sur Audiobell

Les paramètres

Audiobell dispose de plusieurs paramètres permettant de régler plus en détail les calculs et l'affichage des différentes notes. Cette section a pour objectif de détailler l'utilisation et les impacts de ces paramètres.

Fréquence du la3/A4

La valeur choisie dans cette liste déroulante va définir la fréquence de base à partir de laquelle toutes les notes seront calculées. Étant donné que les cloches peuvent être de dates très variables, il est possible de choisir une valeur de référence d'une certaine époque afin de garder une cohérence avec de potentielles archives sur la cloche étudiée.

Système de notes

Le système de notes change uniquement leur affichage en se basant sur différentes normes mais n'a aucun impact sur les calculs.

Notes des partiels mesurés

Les partiels mesurés peuvent être calculés selon deux méthodes distinctes. La méthode "absolue" calcule chaque note à partir de la fréquence du la3/A4 choisie (comme pour les partiels théoriques). La méthode "relative" calcule chaque note par rapport aux notes des partiels théoriques.
Prenons comme exemple un fondamental théorique de 659 Hz, un fondamental mesuré à 711 Hz et un la3/A4 à 440 Hz.
Avec la méthode absolue on obtient un mi4 parfait en note théorique et un fa4 (+5/16) en note mesurée. Avec la méthode relative, la note mesurée est calculée à partir de la note théorique ce qui donne ici, un mi4 (+21/16) pour la note mesurée.

Sauvegarder les résultats

Une fois les calculs effectués, Audiobell permet de télécharger les résultats sous différentes formes afin de les exploiter ultérieurement. Le format "image" sauvegarde les résultats sous forme d'image PNG comme affichés sur la page. Le format "tableau de données" permet de télécharger un fichier CSV exploitable par un logiciel tableur, contenant les résultats actuels. Notez que toutes les valeurs doivent être renseignées pour pouvoir utiliser cette option.

Graphique

Il est possible de générer un graphique permettant de visualiser les différents partiels en fonction de la norme de Limbourg. Ce graphique peut ensuite être téléchargé au besoin (en différentes tailles) au format PNG. Notez que toutes les valeurs doivent être renseignées pour pouvoir utiliser cette option.

Aide au calcul des notes

L'outil "Notes parfaites les plus proches" simplifie le remplissage des partiels théoriques.

Fréquence mesurée

Il est nécessaire de remplir en premier lieu la fréquence mesurée (à l'aide d'un outil externe) de l'un des 5 partiels du son de cloche.
Une fois la fréquence renseignée, les informations "Note précédente" et "Note suivante" seront remplies avec les fréquences des deux notes parfaites les plus proches de la fréquence mesurée.
De plus, la note parfaite la plus proche parmi les deux sera indiquée dans "Fréquence mesurée".
En cliquant sur "Utiliser", il est ensuite possible d'utiliser la note la plus proche choisie ainsi que la fréquence mesurée (en tant qu'un des 5 partiels) afin de remplir tous les partiels théoriques.

Le fonctionnement

Après avoir réglé les différents paramètres, les premières valeurs à renseigner sont les partiels théoriques. Avec Audiobell, la saisie est facilitée puisque tous les partiels théoriques sont automatiquement calculés en fonction de la dernière modification.
Une fois toutes les valeurs théoriques correctes, il est possible de renseigner les partiels mesurés afin de voir les résultats s'afficher et de pouvoir les exporter.
